Block 2: Root-Finding Function

In this question you will use an alternative algorithm to find the square root of a number called the secant method. This algorithm can find the positive number y that satisfies y^2=x for any x\geq0.

You will write a Python function that takes two inputs:

  1. x, the number we want to take the square root of. This will be denoted by x in our code.
  2. \epsilon_{tol}, the tolerance level. This will be denoted by tol in our code.

The function will return the positive number y that approximately solves y^2-x=0. In other words, it will find a y such that |y^2-x|<\epsilon_{tol}, where \epsilon_{tol} is a small positive number.

The steps of the algorithm are as follows:

  1. Begin with initial guesses y_0=0 and y_1=1 for y and set n=1.
  2. Update the guess y_n according to: y_{n+1}=y_n- \frac{\left(y_n-y_{n-1}\right)f(y_n, x)}{f(y_n, x)-f(y_{n-1}, x)} where f(z, x)=z^2-x.
  3. If the distance |y_{n+1}-y_n| > \epsilon_{tol}, increment n by 1 and go back to step 2. Otherwise set y=y_{n+1} and return y.

Use the following template script to write the function findRoot(x, tol) and to answer the questions below: sample-block-2-template.py. Note: using this template is mandatory; failure to do so will result in zero points for this block.

If you completed writing the findRoot(x, tol) function correctly, round(findRoot(2, 1e-6), 8) should return 1.41421356.

  1. [4 pt] Complete writing the function. Use the function to calculate the square root of x=7 with a tolerance level \epsilon_{tol}=0.01. Print the output of findRoot(7, 0.01) in your script.

Answer:

def f(z, x):
    return(z ** 2 - x)

def findRoot(x, tol):
    # Set initial guess:
    (y_0, y_1) = (0.0, 1.0)

    # Initialize distance:
    dist = 1 + tol

    while dist > tol:

        # Compute y_2 according to the equation in step 2
        y_2 = y_1 - (y_1 - y_0) * f(y_1, x) / (f(y_1, x) - f(y_0, x))

        # Compute distance:
        dist = abs(y_2 - y_1)

        # Replace y_0 with y_1 and y_1 with y_2
        (y_0, y_1) = (y_1, y_2)

    return(y_1)

# Test the function using test values provided:
print(round(findRoot(2, 1e-6), 8))

# Calculate the square root of x=7 with tol=0.01:
print(findRoot(7, 0.01))

  1. [3 pt] Add a method that returns the area of a triangle, which is given by the formula \frac{1}{2}|x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)| where |\cdot| denotes the absolute value. Compute and print the area of the triangle in the previous questions using your method. The answer should be 3.

Answer:

class Triangle:
    def __init__(self, corner1, corner2, corner3):
        self.corner1 = corner1
        self.corner2 = corner2
        self.corner3 = corner3
    
    def centroid(self):
        c_x = (self.corner1[0] + self.corner2[0] + self.corner3[0])/3
        c_y = (self.corner1[1] + self.corner2[1] + self.corner3[1])/3
        return [c_x,c_y]
        
    def area(self):
        part1 = self.corner1[0]*(self.corner2[1] - self.corner3[1])
        part2 = self.corner2[0]*(self.corner3[1] - self.corner1[1])
        part3 = self.corner3[0]*(self.corner1[1] - self.corner2[1])
        return 0.5*abs(part1+part2+part3)
    
triangle1 = Triangle([1,1],[1,4],[3,3])
print(triangle1.area())
3.0
